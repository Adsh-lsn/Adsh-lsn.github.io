左斜杠和右斜杠的使用场景

发表于 2021-04-19 分类于 java

  1. 在cmd命令提示符中：

    \右斜杠： 表示路径分隔符，比如：cd F:\Java\jdk-12.0.2\legal
    / 左斜杠：表示命令中加参数的分隔符，比如cd ../..可以实现返回上2级目录。

  2. 在window系统中，文件路径既可以是/，也可以是\。

    F:/Java/jdk-12.0.2/legal √

    F:\Java\jdk-12.0.2\legal √

  3. 在Linux或Unix系统中，文件路径只能是/。

  4. web中的网址跟linux一样，用/。

  5. 在java程序中，文件路径可以是\,因为在Java的语法中\代表转义，所以必须是两个\，才能表达路径中的\。其实，java程序中用/也是可以的。

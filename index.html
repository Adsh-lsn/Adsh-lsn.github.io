娱乐至死

发表于 2021-04-17 分类于 书籍

评分：4颗星

​ 《娱乐至死》是一本不错的预测未来关于电子信息媒体对于人们生活的影响的书，前半部分将印刷术，没有引起我的很大兴趣，后半部分讲到电报、海报、电视机，觉得讲的挺有意思的，可能更贴近于生活。

​ 这本书给我留下的最深刻的印象就是：在印刷体时代，即文字作为信息主要的载体，进行信息的传播，这种方式让读者养成了更加深入理性，逻辑性强的特点，伴随着信息的形式的变化，该形式所表现得内容受制于该形式，比如，报纸时代的到来，使信息变成了一种商品，而且，信息在逐渐变小，没有前后文，信息与信息之间也没有联系，在人们获取信息更加便捷的同时，人们的理解力和耐心正在逐渐降低。看看现在的抖音，由于是短视频，所以更是大大限制了内容的质量。我同学告诉我，他现在都没有耐心去好好去看一部电影，静不下心来，在如此快餐化的信息中，我们有记住了什么，我们又学会了什么。书中提了一个有趣的观点，就是信息——行动，这个比例正在逐渐减少。以前，我们获取信息，这个信息会让我们采取行动，但是现在呢，铺天盖地的消息，我们一条一条的划过去，在我们的脑海中留下了什么呢？对我们的生活有什么启示意义吗？似乎都没有。所以呢，信息——行动的比例就可以成为我以后衡量一个信息的标准。举个最近发生的信息——行动的例子，我在博雅群里看到了有个人说：她每次都要点一点京东价保，因此，我就研究了京东如何进行价保，在我买的两件东西中进行了价保，就便宜了13元。这就是一个很好的例子，跟我们的生活和利益息息相关的，才是一个好的信息，从此，我要开始甄别没用的信息，养成一种有意识的行为，下一步就是多看一些有用的信息。多看一下书本，少看一些片段性的东西，越是片段性的事物，越容易分散我们的注意力，越容易断章取义。在以后看新闻时，也要谨防新闻的断章取义。有时候差一个字，意思就是天壤之别。例子：英文期刊上写的武汉的病毒，结果翻译过来成为了武汉病毒，造成了很大的误会，这个误会可能是无意为之，也有可能是有意为之，如果我们听了这样的言语，岂不是也成了谣言的传播者了吗？所以，以后谨言慎行，因为片段性的信息具有迷惑性。

​ 书中有一句话——到处是水，却没有一滴水可以喝。

​ 始终保持理性和思考。

如何理解java的跨平台

发表于 2021-04-13 分类于 java
  1. 跨平台中的平台指的是操作系统，跨平台就是指可以在多个操作系统上运行（Mac 系统，window系统，Linux系统）。
  2. 跨平台的基础就是有JVM（java Virtual Machine），JVM就是一台虚拟的计算机，java虚拟机屏蔽了一些和操作系统相关的一些信息，比如：各种操作系统有不同的API，ABI，CPU字节（32位，64位），只要编译一次生成class字节码文件，该字节码文件就可以在不同的操作系统上运行，这就是所说的，一次编译，多次运行。（可以屏蔽操作系统和CPU的不同）
  3. java的跨平台性是针对C而言的，java需要安装运行环境，而C不需要安装运行环境。

疑问？

现在流行的系统有windows，Linux,Mac OS，但是Mac OS官方明确不允许java，只是实现了就两个系统上跨平台，JVM带来的性能损失也比较多，那java的最大的特性还是有必要的嘛？

如何从github下载项目或者单个文件夹或者文件

发表于 2021-04-13 分类于 java

  1. 下载整个项目

    在任何一个项目首页页面，会有Code选项，然后选择“Download ZIP”.

    如何提高zip压缩文件的下载速度，可以用一些代理网址，比如：https://d.serctl.com

    在DownLoad Zip右键选择复制地址，然后输入到代理网址的输入框中。800M的内容需要7min左右，已经够快了。

  2. 下载文件夹

    借用一个网站：kinolien.github.io/gitzip/

    注意：每一次都要生成新的token命令。

  3. 下载文件

    右键点击“raw”，然后选择“从链接另存文件为“

注意：

  • 当使用github时，需要打开vpn
  • 如果不使用时，就关闭，否则打开其他网页变慢

2021年4月第一周周总结

发表于 2021-04-12 分类于 总结

博客：5篇

书：3本，《被讨厌的勇气》，《你的善良必须多点锋芒》，《说话的魅力》

与人相处之道

1.对于一件事物感到奇怪，不要立刻做出判断，多思考几种情况，然后找机会验证自己的判断，然后通过最后的判断反推出为什么不是其它种可能，进而可以分析人的性格，因为正是由于具备这种性格，才会做出这样的选择。讲事情的情况考虑的越多，越全面，越有助于自己做出正确的判断，这有一个基本的前提，对人性和对事物有一个基本的认知。

2.要沉得住气，不能因为对方不接电话，不回消息，就乱了自己的阵脚，只要情况还在自己的控制范围以内，就不用一直打电话，一直发消息，你越主动，其实越被动。

3.做任何事情，心里要清楚，这件事情真正的结束是在什么时候，要把事情有头有尾的进行，不能干到一半就扔那不管了，任何东西用完都要养成归位的习惯。比如：每次吃完饭，不算结束，还有刷碗，搞卫生；每次洗完澡不算完事，还有摸身体乳，洗内裤和洗内衣；我现在离开学校不算真正的结束，还有户口，档案，团员关系，把这些未完成的事情，都放到滴答清单里，时不时地看一眼；只有把所有的流程都做完了，才算是真正的结束。在做每件事情，都要思考，准备工作是什么，收尾工作是什么，这样每件事情都有一个好的结局，才不会给未来的自己留下坑。

4.要勇于表达出自己的想法，我大伯帮我找人找工作，但是我还没准备好，就直接告诉我大伯，我需要在准备一段时间。我知道，我说这句话的时候，我大伯和我大伯帮我找的人会对我的能力有看法，但是，我不能明知道自己不行，还硬上，这可能浪费别人的情感。现在的我，比以前更能与压力和焦虑相处，可能正因为经历过更痛苦的岁月，觉得自己可以克服现在找工作的困难。

技术上的进步

1.开始用自己搭建的博客写一些博客，记录自己的思想和成长。

2.博客上能上传图片了。

3.能熟练的新建博客，写博客，上传博客了。

这星期有点少，下星期努力。

选择和判断

1.9点到4点半办理退宿，我以为中午也可以办理，结果12点到2点休息。判断失误。

判断依据：

  • 银行中午有值班的——>物业也应该有值班的，因为他们都是事业单位。
  • 本来就开放3天，应该最后一天人多，所以应该中午开放。

判断失误原因：

  • 虽然学校物业和银行都是属于事业单位，但是物业面对的是学生，人数少，银行面对的是社会人员，人数多，学生是弱势群体，所以，他们应该不会太顾及我们。
  • 我们是快要走的人，跟他们没关系了，物业从我们这也赚不到钱了，所以，也不必在乎我们了，如果是收学费，估计他们中午也值班吧。

经验：

  • 下次提前一天咨询工作时间，做好准备工作。

2.导师给我发两个招聘信息，一个是天津工业大学的，另一个是天津中医药大学附属医院的。判断：老师故意给我发，目的是让我主动找他。

判断依据：

  • 前面没有任何铺垫的话，后边也没有说明，只有链接。老师就是为了化被动为主动。
  • 按照老师之前一贯的风格，他先给你说的，都不是重点，重点都在后边。
  • 老师从不替我们考虑，如果真伪我们的就业考虑，从去年7月份就应该让我们好好写大论文，找工作，尽可能给我们时间，而不是现在惺惺作态。

结论：未得到证实

京东报价省钱策略

发表于 2021-04-12 分类于 省钱策略

​ 灰哥叔叔的给孩子的商业周刊中的微信群中有人提到了，他每次在京东买东西都会时不时的查看价格保护，买机票要用最破的手机买，机票价格更便宜。

​ 从此，开始了我的价格保护之路，看看我的旗开得胜吧。

​ 周口牛肉，购买价148，几天后的价格为139.6，申请保价后退还8.4元。

​ 音响，购买价49.9，第二天价格为44.9，申请保价后退还5元。

申请价格保护步骤：

  1. 打开京东
  2. 最下方最右侧，点击“我的”
  3. 点击中间的“客户服务”
  4. 点击“自助服务”下的“价格保护”
  5. 选择要保价的商品即可。如果商品价格未变化，则不进行保价；如果价格降低，则会保价成功，差价回退还到微信钱包里。

假面山庄

发表于 2021-04-12 分类于 书籍

评分：4星

​ 《假面山庄》是日本作家东野圭吾写的一篇悬疑小说，剧情流畅，我应该是一口气看了3个多小时，内容不长，但是引人入胜。

​ 故事就是局中局，朋美是女主角，出了车祸去世了。彭梅的父亲申彦设了一个局，将男主高之和朋美的母亲、哥哥、表妹和高中同学，申彦的女秘书聚集在一起，自导自演了一部抢了银行的劫匪绑架了这些人的故事。在被绑架的2至3天里，又出现了很多奇异的事件，比如：门口的sos被人用水洗去，计时器被破坏，表妹被杀，表妹的日记本被撕掉，由这些信息，大家推测出杀死表妹的凶手是朋美的父亲申彦，申彦知道表妹喜欢自己女儿的未婚夫，觉得是表妹将朋美的止疼药换成了安眠药，才导致开车过程中睡着了，导致坠入悬崖。申彦发现自己杀人的事情被所有hexo人知道了，就选择了从窗户跳入湖中。晚上的时候，申彦又回来找到了高之，然后告诉了高之，他发现表妹没有杀害朋美，但是表妹说，她也是同罪，意思就是表妹在包庇某个人。表妹当时在和朋美吃饭时，发现表妹吃的止疼药并不是平时吃的止疼药，而是形状相似的安眠药，表妹将自己的止疼药给了朋美。如果是这样，朋美就吃的是止疼药，而不是安眠药，那为什么朋美还会开车坠入悬崖？检查遗物的时候，朋美随身携带的只能装2片药的药盒，里面还有2片药。这样的话，只要证明那2片药是安眠药，就可以证明朋美不是自杀。但是，检查时发现，药盒里的两片药就是止痛药。父亲说，当时有目击证人看见，朋美下车走到悬崖边，来回走了好几趟，最后开车冲了下去，所以，是自杀。而事实的真相就是，在表妹说出药不一样的时候，就知道是未婚夫高之把药换了，她没有吃表妹给她的止痛药，而是把安眠药扔了，把表妹的安眠药放进了药盒里，即使死去了，也不愿意别人发现是未婚夫要害她。而未婚夫要害她的原因就是他喜欢上了她的表妹，想跟表妹在一起，但是又由于在生意上要依靠朋美的父亲，所以无法直接悔婚。在偶然间发现了止痛药和安眠药一样，就心生恶意，高之也很犹豫，既想让朋美死，也不希望她死去。而朋美选择自杀，一方面是绝望，另一方面是成全吧！

​ 总之，我是看进去了，整个画面感很强，希望有空，再读一遍。

说话的魅力

发表于 2021-04-10 分类于 书籍

评分：3颗星

书中给了好多以前没有注意到的细节，平时应该多观察，多积累。

  1. 起名字时，最后一个字是一二声，会显得比较洪亮。
  2. 到领导办公室，需要坐在离门口最近的位置，做好随时走的准备，而且不会给领导带来压迫感。
  3. 演讲时，要注意幕布的颜色，如果幕布是深颜色，就要穿浅颜色；如果幕布是浅颜色，要穿深颜色。这些都需要提前做好功课。
  4. 如果是领导，对椅子也有要求，靠背不能太高，显得自己太矮。
  5. 说话的时候，身体往前倾，会显得有诚意、如果在坐着，椅子也要随着身体转动，不能身子转过去了，椅子没动。
  6. 如果想赢在起跑点上，就要在别人看不到的地方下功夫，给人说不出的一种好感。
  7. 男生面试，不能穿全新的衣服，女生面试要穿新衣服。
  8. 努力，是为了产生一种信仰，使你有信心，有勇气，有气度。

总结

总而言之，可以在重复看一遍，确实有很多我们需要注意，但是如果没有人提醒，我们就会忽略的细节。看了这些，我们可以做的更周到。

如何判别好人

发表于 2021-04-10

毕业季，闲置东西大赠送如下：

顺丰快递员：一盒乒乓球。

京东快递：绝对小孩。

学弟：阿道夫洗头液，一袋滋源，一袋蓝月亮，半盒眼镜布。

生哥：一个桌垫，悦诗风铃洗面奶，一支百乐笔，一盒青草膏，一支蚊虫叮咬膏。

方便面女孩：两个收纳袋，无印良品水，无印良品乳，一盒青草膏，一套课桌椅。

5层打扫卫生阿姨：一袋滋源洗发液，阿道夫护发素，小黑裙沐浴乳，一个收纳盒，3个吃饭盒。

收获

顺丰快递员：**给了我优惠，本来价格在70元，最后经过优惠才支付了57元。

京东快递：收获了快递员开心的真诚的笑。

学弟：请我吃了博格达

生哥：一直要修改我的简历

方便面女孩：鲜虾泡面（2鸡蛋，一根火腿）；粗粮面（5种丸子，每种3个）；土豆粉（好多丸子）

5层打扫卫生阿姨：河南周口人

总结：

  1. 懂得感恩的人，多交往；不懂感恩的人，不深交。
  2. 要多观察，一个眼神一个动作，就能知道对方是什么人。方便面女孩见到我时的立刻转身；卫生阿姨的微笑；别人搬东西时的手酸手痛，依旧坚持；一直跟我传授面试经验的生哥；不让我声张的顺丰快递。
  3. 因为小时候家里没钱，导致对物质的一种强烈渴望。过去一直活在羡慕别人的状态里，现在什么东西都想自己拥有，而且还有屯的习惯。既然知道了导致自己买东西的诱因，以后就可以稍加克制。告诉自己，拥有很多东西是一种负担，东西越少越少，而且，在买东西之前，查看自己是否有重复的东西或者可替代的东西。如果，别人有而且自己不常用，就借用别人的。如果不是贴身物品或者电器，可以通过咸鱼购买。如果是贵重物品，自己可能用的频率比较高，那就延迟一周再买。不要因为双十一各种活动，就买买买，看好自己的钱包。

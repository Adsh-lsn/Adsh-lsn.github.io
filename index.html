电脑中多个jdk应该如何切换不同版本

发表于 2021-04-20 分类于 java

电脑中是可以安装多个不同版本的jdk的，用到那个版本的jdk，只需要更改对应的JAVA_HOME对应的路径即可。

现在jdk的版本是12，将jdk版本修改成8

修改前：

  1. jdk版本为12
  2. 环境变量为：F:\Java\jdk-12.0.2

修改：

  1. 选择浏览目录，选择安装jdk8的根目录：F:\Java\jdk1.8.0_202
  2. 点击确定。

修改后：

jdk版本变为8

由于环境变量Path中有对JAVA_HOME的引用，所以只需要修改JAVA_HOME的值，Path的值会自动修改。

2021年4月第二周周总结

发表于 2021-04-19 分类于 总结

博客：9篇

书：1本，《娱乐至死》

与人相处之道

1.在多次看房过程中，我可能比较挑剔，因为经历过之前的教训。房东给我的评价是第一次见到看房看的这么仔细的。现在的这个房东也应为当时提了很多意见，能察觉到不高兴。但是，在住进来后发现，有自己的要求是对的，而且要在一开始就提出来，不能一开始就相互忍受。也不是所有的都要提出来，但是，自己不能接受的，自己一定要提出来，否则，以后都会成为隐藏的雷。

2.一旦对方觉得吃亏了，就一定对你更加挑剔。比如：秋鼎园的房东，一直觉得租给我2个月，每个月920，很便宜了。如果不是她的妯娌，不会这么便宜租给我。正是因为她觉得她吃亏了，所以，一定要在其它地方找补回来，比如：煤气一个月多加20，如果朋友来住另算钱。所以，以后再为人处世中，不要要别人觉得不平衡，平衡才是关系融洽的核心。

3.自己从小和别人的生活条件差很多的情况下，依旧可以考上大学，考上研究生，那么我的身上肯定有一些过人之处。比如：无诱惑，无欲望，线性思维，认真，踏实。这些是我在没有太多管教的情况下，能考上高中的竞赛班，大学的卓越班，研究生的重要特质。但是，虽然考上了研究生，但是自己各方面适应的更不是很好，尤其是人际关系，给我带来了很多负面影响。这就说明，紧靠这些是无法面对现在这么复杂的人性和世界，这也对我提出了更多更高的要求，比如：要从线性思维变成发散性思维，要经受住一些（吃喝玩乐）诱惑，要变得有欲望（挣钱）。认真，踏实将是我一生工作的准则，但是，对待人际关系方面脑子也要学会灵活和变通。

4.不要随便下结论，比如：你肯定嫁不出去，你肯定过不好等等

不要随便讨厌一个人一件事，要尝试着理解，比如：为什么王祎婧这么会拒绝人，每次她拒绝我时，应该想想她是怎么拒绝我的，我被拒绝后心里难过吗，我以后可以怎么拒绝别人，而不是对于被拒绝这件事情那么抵触那么讨厌，甚至讨厌这个人。

不要随意否定别人的想法，要多听一听对方是怎么想的，自己是否只是考虑的片面了一些。

技术上的进步

1.开始边听课，边记录一些不懂得问题，有时候，老师只是讲了一个知识点的一部分，所以，需要自己把剩余的部分补齐，这种方式的优点就是调动了自己学习的积极性，缺点就是花费了很多的时间。

2.将一直困扰自己的左斜杠和右斜杠问题解决了，以后遇到这些问题，再也不会抓瞎了。

3.从github上下载整个项目很容易，但是下载项目下的某个文件夹这个问题让自己头疼，现在，一次性的把如何下载项目，文件夹和文件问题一并解决了，开心。

选择和判断

1.中介豆凯（河北邢台人），经介绍，住进了仁德里10号楼。由于东西多，他就开了一辆很破旧的面包车帮我搬家，搬完家后的第二天，发现丢失了一个京东小盒子，里面装着刚买的音响。问了他3遍，每次都说，等等，但是一直没有消息。

判断依据：

  • 嫌给的红包少
  • 不想跟我有联系，觉得我很事。
  • 很忙，没有空。
  • 东西私吞了，不想还回来。

结论

  • 下次见面了，当面问一问，证实一下自己的猜想。

注意：

​ 1、”C:\Program Files\Java\jdk-12”该文件夹地址是JDK安装位置
​ 2、从JDK 9开始发生重大变化 - - - “之前类和资源文件存储在lib/rt.jar， tools.jar，JDK 9版本开始 lib/dt.jar和其他各种内部JAR文件都存储在一个更有效的格式在实现特定的文件lib目录。所以不需要配置CLASSPASTH变量。”

​ 3.jdk7和8在安装完jdk后，会自动安装jre，但是在jdk之后的版本，比如jdk12，需要手动安装。

五、检查JDK配置

​ 键盘 “Win+R” 快捷键键调出“运行”，输入“cmd”，在命令提示符输入“java -version”

目录名称 目录作用
bin 包含一些用于开发Java程序的工具，例如：编译工具(javac.exe)、运行工具 (java.exe) 、打包工具 (jar.exe)等。
conf 配置文件
include C语言的头文件，用于支持Java程序设计。
jmods 该路径放了JDK的各种模块，可以防止反射查看模块的内部内容，其实对模块起到了一个保护的作用。
jre Java运行时环境的根目录。
jre\bin 包含Java平台所使用工具和类库的可执行文件和DLL文件。
jre\lib Java运行时环境所使用的核心类库。
legal 该路径存放了JDK各模块的授权文档
lib 包含开发Java程序所使用的类库文件。

左斜杠和右斜杠的使用场景

发表于 2021-04-19 分类于 java

  1. 在cmd命令提示符中：

    \右斜杠： 表示路径分隔符，比如：cd F:\Java\jdk-12.0.2\legal
    / 左斜杠：表示命令中加参数的分隔符，比如cd ../..可以实现返回上2级目录。

  2. 在window系统中，文件路径既可以是/，也可以是\。

    F:/Java/jdk-12.0.2/legal √

    F:\Java\jdk-12.0.2\legal √

  3. 在Linux或Unix系统中，文件路径只能是/。

  4. web中的网址跟linux一样，用/。

  5. 在java程序中，文件路径可以是\,因为在Java的语法中\代表转义，所以必须是两个\，才能表达路径中的\。其实，java程序中用/也是可以的。

娱乐至死

发表于 2021-04-17 更新于 2021-04-19 分类于 书籍

评分：4颗星

　　《娱乐至死》是一本不错的预测未来关于电子信息媒体对于人们生活的影响的书，前半部分将印刷术，没有引起我的很大兴趣，后半部分讲到电报、海报、电视机，觉得讲的挺有意思的，可能更贴近于生活。

　　这本书给我留下的最深刻的印象就是：在印刷体时代，即文字作为信息主要的载体，进行信息的传播，这种方式让读者养成了更加深入理性，逻辑性强的特点，伴随着信息的形式的变化，该形式所表现得内容受制于该形式，比如，报纸时代的到来，使信息变成了一种商品，而且，信息在逐渐变小，没有前后文，信息与信息之间也没有联系，在人们获取信息更加便捷的同时，人们的理解力和耐心正在逐渐降低。看看现在的抖音，由于是短视频，所以更是大大限制了内容的质量。我同学告诉我，他现在都没有耐心去好好去看一部电影，静不下心来，在如此快餐化的信息中，我们有记住了什么，我们又学会了什么。书中提了一个有趣的观点，就是信息——行动，这个比例正在逐渐减少。以前，我们获取信息，这个信息会让我们采取行动，但是现在呢，铺天盖地的消息，我们一条一条的划过去，在我们的脑海中留下了什么呢？对我们的生活有什么启示意义吗？似乎都没有。所以呢，信息——行动的比例就可以成为我以后衡量一个信息的标准。举个最近发生的信息——行动的例子，我在博雅群里看到了有个人说：她每次都要点一点京东价保，因此，我就研究了京东如何进行价保，在我买的两件东西中进行了价保，就便宜了13元。这就是一个很好的例子，跟我们的生活和利益息息相关的，才是一个好的信息，从此，我要开始甄别没用的信息，养成一种有意识的行为，下一步就是多看一些有用的信息。多看一下书本，少看一些片段性的东西，越是片段性的事物，越容易分散我们的注意力，越容易断章取义。在以后看新闻时，也要谨防新闻的断章取义。有时候差一个字，意思就是天壤之别。例子：英文期刊上写的武汉的病毒，结果翻译过来成为了武汉病毒，造成了很大的误会，这个误会可能是无意为之，也有可能是有意为之，如果我们听了这样的言语，岂不是也成了谣言的传播者了吗？所以，以后谨言慎行，因为片段性的信息具有迷惑性。

　　书中有一句话——到处是水，却没有一滴水可以喝。

　　始终保持理性和思考。

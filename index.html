沉默的大多数

发表于 2021-04-22 分类于 书籍

  评分：5分。

  《沉默的大多数》是王小波先生的作品你，他是我很喜欢的一个作家。我在纸上写下了对他的评价——幽默中带有讽刺，有一个有趣的灵魂，但是书中写的观点及其富有逻辑，相互论证推导，举例夸张，但有时有特别能说明问题。关于有些话题看完后，又显得特别沉重。他这个人又是比较直接，不容易改变自己的看法。我很敬佩他这样的人。书中对一些国内不好的现象作了批判，总觉得作了批判是好事，让更多人意识到自身存在的问题，然后再做的更好，否则容易夜郎自大。

  关于善恶的观点，引用其中的话（善于恶为一，正如上坡和下坡是同一条路。不知何为恶，焉知何为善？）

  比较有趣的例子：从此以后，我不敢去看任何一部电视剧，我怕我白发苍苍的老母亲忽然知道自己生了个傻儿子而伤心；就这句话而论，我倒希望它能活过来，说一句话“我是开个玩笑”，然后死去；精美，让读小说的惊喜，让打算写小说的人害怕；至于那道顶冠，既不遮阳，也不挡雨，只能招马蜂进去做窝。

  我个人认为做爱做的事情才是有，做自己也不知道为什么做的事情才是“无”。

  坏事就是坏事，好事就是好事，至于坏事可不可以变成好事，已经是另外一个问题了。

  我认为，人活在世界上，不必什么都知道，只要知道最好的就行了。

  这是因为尊严是属于个人的、不可压缩的空间，这块空间要靠自己来捍卫——捍卫的意思是指敢争、敢打官司、敢动手（勇斗歹徒）。我觉得人还是有点尊严的好，假如个人连个待的地方都没有，就无法为人做事，更不要说做别人的典范。人有无尊严，有一个简单的判据，是看他被当做一个人还是一个东西来对待。这件事有点两重性，其一是别人把你当做人还是东西，是你尊严之所在。

  现在我提起这些事情，绝不是想说这些朴实的人们有什么不对，而是试图说明，人经不起恭维。越是天真、朴实的人，听到一种于己有利的说法，证明自己身上有种种优越的素质，是人类中最优越的部分，就越会不知东西南北，撒起癔症来。

  我们学会了把世上一切事物分成好的和坏的以后，对世界的了解还是非常非常可怜的。我们还要继续学习一切是如何发生、如何变化的。这些知识会冲击我们过去形成的是非标准，这时我们就面临一个重大抉择，是接受事实，还是坚持旧有的价值观念？事实上有很多这样的人：他们“明辨是非”的能力却成了接触世界与了解世界的障碍，结果是终生停留在只会“明辨是非”的水平上。可以这样说，接受了一个伦理的体系不过达到了小学四年级的水平，而接受一个真理的体系就难得多，人们毕生都在学习科学，接触社会。人们知道得越多，明辨是非就越困难。

  王小波先生推荐的几本书：《情人》——王道乾，《青铜骑士》——查良铮（穆旦），《少女与死》，《法官和他的刽子手》，《巴巴拉上校》，《1984》——奥威尔。

  觉得他的文字有种醍醐灌顶的感觉，他有敏锐的洞察力和联想力，能够一阵见血的指出存在的问题，而我一直都置身其中，觉得是很正常的事情，读他的书，有助于更加认识我，我们。无论周围的环境怎么变得负责，但是自己的内心依旧坚定，这或许是我最佩服他的地方吧。

  读完有很多思考，但也有看不懂的地方，有空一定要再看一下。

找工作

发表于 2021-04-22 分类于 我

  自从20121年4月10号开始，我就变成了一个无业游民，所有人都催我赶紧找工作，包括：老韩，生哥，我爸，我妈，大伯，兰兰姐，娟姐。给我带来了很大的压力，我就在想，为什么一毕业就要开始上班，我晚两个月上班怎么了。我少挣两个月的钱怎么了，为什么感觉自己是世界上最可恶的人，再也不是最可爱的人了。到了什么时间就该干什么事，但是，能不能从孩子的角度去考虑问题呢？我就是一个挣钱的工具吗？我是个人好不好，我有自己的想法，自己的安排，为什么替我瞎操心，你就是告诉我100遍，投简历，找工作，有用吗？除了给我带了对于的心理负担，让我学不下去，还有什么好处吗？你们认为这个世界上所有的东西都是速成的，但我不认为是这样，人总是要扎扎实实的学，打下坚实的基础，这样以后才轻松。如果我现在去随便找了一份7000至8000的工作，我又能学到什么呢？我对自己高标准严要求有错吗？我可以按照这世俗的约定，上学，工作，结婚，生子。但是，请让我之间有喘息的机会，好吗？我自己有自己的安排，为什么所有人都要来打乱我。

  要学会尊重和相信孩子的想法和看法，而不是一味地按照世俗的观点去要求，我想作为一个人有尊严的活着，也请你们尊重我，我有自己的想法，我不是木偶，我也不想当木偶，我也不会成为木偶。

电脑中多个jdk应该如何切换不同版本

发表于 2021-04-20 分类于 java

电脑中是可以安装多个不同版本的jdk的，用到那个版本的jdk，只需要更改对应的JAVA_HOME对应的路径即可。

现在jdk的版本是12，将jdk版本修改成8

修改前：

  1. jdk版本为12
  2. 环境变量为：F:\Java\jdk-12.0.2

修改：

  1. 选择浏览目录，选择安装jdk8的根目录：F:\Java\jdk1.8.0_202
  2. 点击确定。

修改后：

jdk版本变为8

由于环境变量Path中有对JAVA_HOME的引用，所以只需要修改JAVA_HOME的值，Path的值会自动修改。

2021年4月第二周周总结

发表于 2021-04-19 分类于 总结

博客：9篇

书：1本，《娱乐至死》

与人相处之道

1.在多次看房过程中，我可能比较挑剔，因为经历过之前的教训。房东给我的评价是第一次见到看房看的这么仔细的。现在的这个房东也应为当时提了很多意见，能察觉到不高兴。但是，在住进来后发现，有自己的要求是对的，而且要在一开始就提出来，不能一开始就相互忍受。也不是所有的都要提出来，但是，自己不能接受的，自己一定要提出来，否则，以后都会成为隐藏的雷。

2.一旦对方觉得吃亏了，就一定对你更加挑剔。比如：秋鼎园的房东，一直觉得租给我2个月，每个月920，很便宜了。如果不是她的妯娌，不会这么便宜租给我。正是因为她觉得她吃亏了，所以，一定要在其它地方找补回来，比如：煤气一个月多加20，如果朋友来住另算钱。所以，以后再为人处世中，不要要别人觉得不平衡，平衡才是关系融洽的核心。

3.自己从小和别人的生活条件差很多的情况下，依旧可以考上大学，考上研究生，那么我的身上肯定有一些过人之处。比如：无诱惑，无欲望，线性思维，认真，踏实。这些是我在没有太多管教的情况下，能考上高中的竞赛班，大学的卓越班，研究生的重要特质。但是，虽然考上了研究生，但是自己各方面适应的更不是很好，尤其是人际关系，给我带来了很多负面影响。这就说明，紧靠这些是无法面对现在这么复杂的人性和世界，这也对我提出了更多更高的要求，比如：要从线性思维变成发散性思维，要经受住一些（吃喝玩乐）诱惑，要变得有欲望（挣钱）。认真，踏实将是我一生工作的准则，但是，对待人际关系方面脑子也要学会灵活和变通。

4.不要随便下结论，比如：你肯定嫁不出去，你肯定过不好等等

不要随便讨厌一个人一件事，要尝试着理解，比如：为什么王祎婧这么会拒绝人，每次她拒绝我时，应该想想她是怎么拒绝我的，我被拒绝后心里难过吗，我以后可以怎么拒绝别人，而不是对于被拒绝这件事情那么抵触那么讨厌，甚至讨厌这个人。

不要随意否定别人的想法，要多听一听对方是怎么想的，自己是否只是考虑的片面了一些。

技术上的进步

1.开始边听课，边记录一些不懂得问题，有时候，老师只是讲了一个知识点的一部分，所以，需要自己把剩余的部分补齐，这种方式的优点就是调动了自己学习的积极性，缺点就是花费了很多的时间。

2.将一直困扰自己的左斜杠和右斜杠问题解决了，以后遇到这些问题，再也不会抓瞎了。

3.从github上下载整个项目很容易，但是下载项目下的某个文件夹这个问题让自己头疼，现在，一次性的把如何下载项目，文件夹和文件问题一并解决了，开心。

选择和判断

1.中介豆凯（河北邢台人），经介绍，住进了仁德里10号楼。由于东西多，他就开了一辆很破旧的面包车帮我搬家，搬完家后的第二天，发现丢失了一个京东小盒子，里面装着刚买的音响。问了他3遍，每次都说，等等，但是一直没有消息。

判断依据：

  • 嫌给的红包少
  • 不想跟我有联系，觉得我很事。
  • 很忙，没有空。
  • 东西私吞了，不想还回来。

结论

  • 下次见面了，当面问一问，证实一下自己的猜想。

开心的

  1. 自己对事物的判断准确。比如：推测出卫生间的拖把是隔壁的而不是阿姨拿走的。
  2. 做了正确的选择。比如：为了离开装修噪声源选择搬家，付出了3天忙碌的代价，但是是值得的。
  3. 自己进步了，自我提升了，攻克难题了，用自己的智慧解决了问题。
  4. 跟X老师没有联系了。
  5. 跟鸭鸭彻底断绝联系，干了一件一直想干，没有干成的事情，爽。在爽后，要思考自己做了什么，让事情一步一步走到这样。
  6. 后续满满补充。。。

难过的

  1. 被别人忽略，区别对待。比如：我去红姐那，红姐从来不拖地，而当点点去红姐那，红姐就拖了地。

  2. 在很多人的时候插不上话，只能两个人的时候会侃侃而谈，三个人中我就是被剩下的那一个，感觉自己是边缘人。

  3. 不能掌控自己的节奏

  4. 后续慢慢补充。。。

一、JDK下载

​ 1、JDK官网下载地址：https://www.oracle.com/technetwork/java/javase/downloads/jdk12-downloads-5295953.html

3、下载的文件：jdk-12.0.1_windows-x64_bin.exe

注意：

​ 1、”C:\Program Files\Java\jdk-12”该文件夹地址是JDK安装位置
​ 2、从JDK 9开始发生重大变化 - - - “之前类和资源文件存储在lib/rt.jar， tools.jar，JDK 9版本开始 lib/dt.jar和其他各种内部JAR文件都存储在一个更有效的格式在实现特定的文件lib目录。所以不需要配置CLASSPASTH变量。”

​ 3.jdk7和8在安装完jdk后，会自动安装jre，但是在jdk之后的版本，比如jdk12，需要手动安装。

五、检查JDK配置

​ 键盘 “Win+R” 快捷键键调出“运行”，输入“cmd”，在命令提示符输入“java -version”

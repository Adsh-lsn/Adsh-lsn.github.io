无重复字符的最长子串

给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

输入: s = “abcabcbb”
输出: 3
解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

输入: s = “bbbbb”
输出: 1
解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。

输入: s = “pwwkew”
输出: 3
解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。
请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

输入: s = “”
输出: 0

Version1

总结

  1. 要弄清楚各个变量的含义。left为左指针，为开始的字符串的第一个字母。max为无重复字符的最长子串
  2. 熟练掌握哈希表的get()方法和containsKey()方法
  3. 滑动窗口是通过left指针通过跳过不必要的比较实现的。
  4. 注意分类讨论的思想：比如某一个字符在哈希表中，分为2类，一类是该字符在当前字符串中，另一类是该字符不在当前字符串中。

两数相加

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807.

输入：l1 = [0], l2 = [0]
输出：[0]

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]

Version1

思路：

第一种情况：l1不为空，l2不为空

第二种情况：l1不为空，l2为空

第三种情况：l1为空，l2不为空

以下代码前两个示例通过，最后一个没通过。

Version2：官方答案

执行用时：2 ms, 在所有 Java 提交中击败了100.00% 的用户

内存消耗：38.9 MB, 在所有 Java 提交中击败了47.67% 的用户

总结

  1. 如果出现一个链表为空，另一个链表不为空，可以将空的地方都设置为0，方便操作。
  2. 将（l1不为空）和（l1为空，但l2不为空）两种情况通过或||运算符进行整合操作。
  3. 注意最后最高位相加，产生进位的情况，要在创建一个节点，保存最高位。

LeetCode之两数之和

发表于 2021-05-02 更新于 2021-05-03 分类于 leetcode

两数之和

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

输入：nums = [2,7,11,15], target = 9
输出：[0,1]

输入：nums = [3,2,4], target = 6
输出：[1,2]

输入：nums = [3,3], target = 6
输出：[0,1]

Version1

改进：

  1. 将index数组写到if()**方法体中，因为因为该数组在整个代码逻辑中只会用到一次，所以在if()方法体中进行数组的创建和初始化。但是在twoSum()方法的最后一行，添加return new int[0];**返回一个数组引用。
  2. 第9行代码map.put(target-nums[i],i);**应该改成map.put(nums[i],i);。**因为前者将会将很多不属于nums[i]的数放进map中，属于无中生有，会额外增加内存。

Version2

总结：

  1. 数组中同一个元素在答案里不能重复出现，是不是提示了将数组中的元素作为不能重复的key值，而数组下标作为value值。如果是这样，题目要求返回两个数组下标，即返回两个value值。进一步考虑到方法的返回类型就是数组类型int[];
  2. 最后一句return new int[0];**就是返回一个int**数组的引用，数组是空的，这样也可以减少空间。
  3. map.containsKey方法：判断哈希表中是否存在该key值。
  1. map.get方法：通过key值获得value值。
  1. map.put方法：将key值和value值成对的放入哈希表中。

电脑中多个jdk应该如何切换不同版本

发表于 2021-04-20 分类于 java

电脑中是可以安装多个不同版本的jdk的，用到那个版本的jdk，只需要更改对应的JAVA_HOME对应的路径即可。

现在jdk的版本是12，将jdk版本修改成8

修改前：

  1. jdk版本为12
  2. 环境变量为：F:\Java\jdk-12.0.2

修改：

  1. 选择浏览目录，选择安装jdk8的根目录：F:\Java\jdk1.8.0_202
  2. 点击确定。

修改后：

jdk版本变为8

由于环境变量Path中有对JAVA_HOME的引用，所以只需要修改JAVA_HOME的值，Path的值会自动修改。

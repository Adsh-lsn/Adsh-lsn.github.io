说话的魅力

发表于 2021-04-10 分类于 书籍

评分：3颗星

书中给了好多以前没有注意到的细节，平时应该多观察，多积累。

  1. 起名字时，最后一个字是一二声，会显得比较洪亮。
  2. 到领导办公室，需要坐在离门口最近的位置，做好随时走的准备，而且不会给领导带来压迫感。
  3. 演讲时，要注意幕布的颜色，如果幕布是深颜色，就要穿浅颜色；如果幕布是浅颜色，要穿深颜色。这些都需要提前做好功课。
  4. 如果是领导，对椅子也有要求，靠背不能太高，显得自己太矮。
  5. 说话的时候，身体往前倾，会显得有诚意、如果在坐着，椅子也要随着身体转动，不能身子转过去了，椅子没动。
  6. 如果想赢在起跑点上，就要在别人看不到的地方下功夫，给人说不出的一种好感。
  7. 男生面试，不能穿全新的衣服，女生面试要穿新衣服。
  8. 努力，是为了产生一种信仰，使你有信心，有勇气，有气度。

总结

总而言之，可以在重复看一遍，确实有很多我们需要注意，但是如果没有人提醒，我们就会忽略的细节。看了这些，我们可以做的更周到。

如何判别好人

发表于 2021-04-10

毕业季，闲置东西大赠送如下：

顺丰快递员：一盒乒乓球。

京东快递：绝对小孩。

学弟：阿道夫洗头液，一袋滋源，一袋蓝月亮，半盒眼镜布。

生哥：一个桌垫，悦诗风铃洗面奶，一支百乐笔，一盒青草膏，一支蚊虫叮咬膏。

方便面女孩：两个收纳袋，无印良品水，无印良品乳，一盒青草膏，一套课桌椅。

5层打扫卫生阿姨：一袋滋源洗发液，阿道夫护发素，小黑裙沐浴乳，一个收纳盒，3个吃饭盒。

收获

顺丰快递员：**给了我优惠，本来价格在70元，最后经过优惠才支付了57元。

京东快递：收获了快递员开心的真诚的笑。

学弟：请我吃了博格达

生哥：一直要修改我的简历

方便面女孩：鲜虾泡面（2鸡蛋，一根火腿）；粗粮面（5种丸子，每种3个）；土豆粉（好多丸子）

5层打扫卫生阿姨：河南周口人

总结：

  1. 懂得感恩的人，多交往；不懂感恩的人，不深交。
  2. 要多观察，一个眼神一个动作，就能知道对方是什么人。方便面女孩见到我时的立刻转身；卫生阿姨的微笑；别人搬东西时的手酸手痛，依旧坚持；一直跟我传授面试经验的生哥；不让我声张的顺丰快递。
  3. 因为小时候家里没钱，导致对物质的一种强烈渴望。过去一直活在羡慕别人的状态里，现在什么东西都想自己拥有，而且还有屯的习惯。既然知道了导致自己买东西的诱因，以后就可以稍加克制。告诉自己，拥有很多东西是一种负担，东西越少越少，而且，在买东西之前，查看自己是否有重复的东西或者可替代的东西。如果，别人有而且自己不常用，就借用别人的。如果不是贴身物品或者电器，可以通过咸鱼购买。如果是贵重物品，自己可能用的频率比较高，那就延迟一周再买。不要因为双十一各种活动，就买买买，看好自己的钱包。

你的善良必须多点锋芒

发表于 2021-04-10 分类于 书籍

评分：2.5星

​ 知道这本书很长时间了，自己也买了纸质书，但是一直没有阅读。我是因为名字吸引了我，后来才知道，你的善良必须有点锋芒，只是其中的一个小故事。作家已经从事本专业10余年，但是，读起来，确实不费劲，讲的都是他自己，他的朋友，他的读者亲身经历的一些事情。有些事情也能从中获取一些共鸣，但是最后一章就恋爱的，就读下去了，感觉谈过恋爱的都知道，讲的东西浅显易懂。

​ 讲讲其中的收获吧。

  1. 对于人生中最重要的就是工作和和谁一起生活。工作就是自己热爱的事情，而挣钱的岗位就是劳动。 和谁一起生活，比如对于伴侣，我们要做到经济独立和精神独立，何为精神独立，没有你，我可以生活的很好，有了你，我活得更好；对于父母，对待他们要给予很多的耐心，引导，注意力。
  2. 不要让自己看起来很努力，要真的很努力，学会坐冷板凳，学会专研，这也是我在研究生生涯学到的最重要的东西。不要一直做准备工作的准备工作。
  3. 如果你的公司是逼着员工加班，哪怕是让你在那熬时间，最好的办法就是尽快离职。
  4. 只有足够重视自己，才会被人重视。
  5. 感谢贵人之前，先感谢自己。
  6. 每个世界都有自己运行的规则，如果只是因为自己的一点怜悯之心随便去打破这个世界的运行规则，那么牺牲的代价远远大于当初施善行为所得到的。

总之，就是这些小道理。

我在想，什么是一本好书，可能给自己启发，让自己有所改变的书吧。

2021年3月30号加兰兰姐微信

2021年4月4号兰兰姐取了快递

2021年4月7号通知兰兰姐取快递（3个快递，一个百世快递，一个京东快递，一个顺丰快递）

2021年4月8号上午11点兰兰姐托姐夫取了快递。

目前，一共8个快递，都在我二舅家，二舅在家可以通风，为了防止东西发霉。

启示：

  1. 尽量少麻烦别人，能让别人跑一趟的，就不跑两趟。
  2. 可以在第二趟的时候，寄给我姐一些吃的，这样我姐再取自己快递的时候，就可以顺带把我的快递取了。

成功：

  1. 在别人没有接自己电话的时候，分析了3种情况（不方便接电话；不是我姐取得快递；我妈和我舅母之间有矛盾了）。事实证明是我姐在做饭，让我姐夫取得快递。
  2. 在我姐没接我电话的时候，我没有再打过去，而是耐心等待，等我姐回过来。什么事情不能太主动，显得自己很着急，要很淡定，至少看起来淡定。
  3. 我给我姐在京东上买了周口的酱牛肉。

后续事情：

  1. 跟踪我姐是否收到了京东快递
  2. 过年回家把我的东西从我二舅家搬到我家，顺便把我妈的东西也一起搬回来（半天时间）
  3. 整理我的东西。（1天时间）

《被讨厌的勇气》

发表于 2021-04-07 分类于 书籍

​ 其实，我一直都知道自己缺乏勇气，勇于改变自己的决心。我呢，一直在维持一种我有很多朋友的繁荣假象，让别人看起来，我并不孤独。这是由于小时候的阴影，无人跟我玩耍，所以，我无意识的就渴望这种被朋友环绕的感觉。也正是这种渴望，促使我一直在讨好别人，只有我讨好别人了，别人才会跟我玩。如今，我已经识破了这种自我欺骗的骗局，我已经不需要这种幻想来弥补自己儿时的遗憾。我认识了自己的过去，知道了现在的原因，我对此也没有很多遗憾，我只知道，我需要改变这种讨好人的模式，甚至下意识的讨好行为。也许，我做出改变了，但是我获得了自由，我不在被他人束缚，我会活得开心。有时候，我们不必他干扰他人的决定，有时候，我们也不能让别人来干涉自己的事情。我们要保持自身的的独立，包括，经济和精神上的独立。然后在探寻与周围人的相处之道，达到一个和谐的氛围。最近，慢慢的，学会如何接别人的话，如何更幽默更有趣的接别人的话，学会慢慢品别人的话，慢慢的发现，现在的我更容易控制节奏，控制自己与他人的关系，慢慢的自己的生活也在变得轻松，或许，这也印证了书中说的，一切的烦恼都是来自于人际关系，人际关系好了，就比较开心。人际关系糟糕，也会影响自己的方方面面。所以，也在慢慢积累自己的人机关系相处经验。

炒圆白菜

发表于 2021-03-06 分类于 -炒菜

材料

  1. 包菜
  2. 蒜
  3. 调味品

步骤

  1. 加入油，油热，调小火，倒入花椒，花椒变色，将花椒铲入到垃圾桶
  2. 将手撕好的包菜，倒入锅中，大火翻炒1min
  3. 倒入醋，酱油，生抽，耗油，香油，继续翻炒半分钟
  4. 倒入切碎的蒜末，出锅

注意事项

不用放盐

租房总结

发表于 2021-03-04 更新于 2021-04-17

租房总结第二版

从空间进行划分

卧室：

  1. 床是否是硬板床，铺床上的被褥是否干净，看一下床是否发霉，是否良好？
  2. 如何晾晒衣服？有无晾衣杆？
  3. 是否有衣柜？多多益善
  4. 有没有电脑桌，电脑桌下方一定要放得下机箱，最好宽一些。
  5. 是否有配套的椅子，椅子一定要是有靠背的。
  6. 坐在椅子上看椅子和桌子的高度是否舒服，如果不舒服，问房东是否可以更换？
  7. 卧室是否有窗帘，窗帘的透光性如何，要厚实一点的。
  8. 卧室有没有纱窗？
  9. 不留多余的东西，如果又不想要的，直接让房东拿走。
  10. 是否有垃圾桶？
  11. 开空调，看空调的制冷和制冷功能是否完好，看空调机箱是否发出很大的噪音？

卫生间：

  1. 有没有热水器？
  2. 水龙头是否是二合一的？
  3. 是不是自动洗衣机，自动洗衣机是否可以用？进水和出水是否方便？
  4. 打开热水器水龙头，看水龙头是否够大，是否有热水？
  5. 看卫生间的镜子干不干净？
  6. 按一下马桶，看出水怎么样？

客厅：

  1. 问房东办理的宽带是多少M的？并自己用手机测试一下

厨房：

  1. 厨房要大一点
  2. 厨房的抽烟机是否好用，不会出现按键个别坏了，抽烟机是否干净？
  3. 打一下煤气罩，蓝色代表充分燃烧，红色代表不充分燃烧。如果冒红火，就说明被堵了，容易让锅底部变黑。
  4. 地面是否容易有积水？厨房桌面是否平整，水龙头旁边会不会漏水？
  5. 打开厨房的小柜门，查看里面是否干净？
  6. 是否有冰箱？

周围环境：

  1. 房东是否大方，扣扣搜搜的不要。
  2. 房东是否爱干净，从厨房到客厅都可以观察到。
  3. 如果和房东一起住，打听一下房东的生活规律，是否有小孩子？
  4. 合租是否是否爱干净，通过卫生间，厨房可以观察到。
  5. 合租室友是女生还是男生？
  6. 周围环境是否嘈杂？有什么噪声源没有?（幼儿园，装修房子，菜市场）
  7. 周围购物、吃饭、乘坐公交是否方便？
  8. 下雨天看房，可以看出小区是否有积水，好不好走？
  9. 白天看房，看光线是否充足，11点看房，听周围声音。

租房总结第一版

应该避免的坑**

  1. 看床是否是硬板床，看床是否有霉味
  2. 看有没有晾衣杆
  3. 看厨房是否够大
  4. 看煤气是否正常，需要打开，看火苗是蓝色的还是红色的。蓝色代表充分燃烧，红色代表不充分燃烧。不充分燃烧的火焰，容易将锅底弄黑
  5. 打开热水器水龙头，看水龙头是否够大，是否有热水。
  6. 看所有的水龙头是分开的，还是二合一，选择二合一的，即装了混合阀的。
  7. 看洗衣机是手动的，还是自动的，选择自动的。
  8. 看房东是否抠门，什么都给你算钱，有朋友来住加钱，做饭用煤气加钱，地暖明明还有20天，收一个月的钱。
  9. 看地面是否干净，东西是否整洁，可以看出房东是不是一个勤快的人。
  10. 看合租的室友，是否爱干净，爱卫生。否则，就不会维护公共卫生。
  11. 看灶台是否干净，上面的油是否擦干净。
  12. 选择在临近中午看房，听周围是否有装修的，或者问房东是否有装修的。顺便看一下房间的通光性怎么样？
  13. 一定要签合同，杜绝房东一直提条件。
  14. 看卫生间有没有窗帘，如果没有窗帘，是否透光？
  15. 是否有一张合适的书桌和椅子，如果没有，是否可以配，不可以配，就不住。
  16. 我的要求，厨房要够大，储物柜要够多，床够硬，房间通风，通亮，周围环境安静。
  17. 观察周围环境，是否有嘈杂源（幼儿园，装修，菜市场）
  18. 公交是否方便。
  19. 尽量选择电梯房，搬家什么的方便。
  20. 窗帘质量如何？
  21. 不要太相信别人。比如前房东，一直说现房东可好了，结果是如此扣扣搜搜的人。
  22. 房子没到期，就不退，否则，双倍加钱。
  23. 不要对陌生人太好。
  24. 网速是否快，问房东多少M网速？
  25. 看卧室有没有纱窗？
  26. 看是否是全自动洗衣机？
  27. 干净、整洁、卫生、安静是第一位的。
